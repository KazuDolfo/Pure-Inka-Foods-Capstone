# Pure-Inka-Foods-Capstone

Proyecto de comercio electrónico para la venta de superalimentos naturales peruanos, desarrollado con un stack moderno de tecnologías web.

## Estructura del Proyecto

- **Backend:** API REST construida con Node.js, Express y MySQL. Maneja la autenticación (JWT), procesamiento de pagos (Stripe), gestión de archivos (Multer) y comunicación en tiempo real (Socket.io).
- **Frontend:** Aplicación SPA desarrollada en Angular 20, con un diseño responsivo y gestión de estado para el carrito y perfil de usuario.

## 🛠️ Tecnologías Principales

- **Frontend:** Angular, Bootstrap, Chart.js, Socket.io-client.
- **Backend:** Node.js, Express, MySQL (mysql2), Stripe SDK, Multer, Bcrypt, JWT.
- **Base de Datos:** MySQL.
